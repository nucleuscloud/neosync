# neosync list

## Overview

Learn how to list Neosync accounts with the neosync accounts list command.

The `neosync accounts list` command is used to list the personal and team accounts for a user or api key.

## Usage

```bash
neosync accounts list
```

### Aliases

```bash
neosync accounts ls
```
